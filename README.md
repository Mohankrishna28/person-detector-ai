# Person Detector AI 👁️

Person Detector AI is a computer vision-based system designed to detect human presence in images or video streams. The project uses deep learning techniques to identify people in real-time and can be extended for surveillance and safety applications.

---
##  Features

- 🧍 Real-time person detection using AI models  
- 🎥 Works with images and video input  
- ⚡ Fast and efficient detection using YOLO-based model  
- 📦 Simple backend processing with Python  
- 🌐 Basic frontend interface for interaction  

---

##  Tech Stack

- **Backend:** Python  
- **Computer Vision:** OpenCV  
- **Model:** YOLOv8  
- **Frontend:** HTML, CSS, JavaScript

2. Run the backend:

3. Open frontend:
- Open `index.html` in browser
